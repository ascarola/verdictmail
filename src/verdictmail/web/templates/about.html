VerdictMail

AI-Powered Email Threat Analysis

v0.2.0
What It Does

VerdictMail is an autonomous email security daemon that monitors a Gmail inbox via IMAP IDLE and runs every incoming message through a multi-stage analysis pipeline in real time — without any user intervention.

Parse Extracts headers, sender identity, originating IP, and all URLs from RFC 822 message bytes.
Enrich Checks SPF, DKIM, DMARC, domain age (WHOIS), DNSBL reputation, display-name spoofing, and expands shortened URLs.
AI Analysis Sends a structured prompt containing all enrichment signals to an LLM (OpenAI, Anthropic, or a local Ollama model) and receives a JSON threat assessment with confidence score and reasoning.
Decide & Act Applies configurable confidence thresholds to produce a final action — pass, flag (IMAP keyword), or move to junk — then logs the full result to a SQLite audit database.

Analysis Pipeline
IMAP IDLE Parse RFC 822 Enrich AI Analyze Decide Audit Log

Each message is processed concurrently in a dedicated thread with its own IMAP connection. The full pipeline typically completes in under 15 seconds with a cloud AI provider.

Deployment

VerdictMail runs as two systemd services on a dedicated Ubuntu 24.04 LXC container hosted in a Proxmox homelab environment:

  • verdictmail.service — the analysis daemon (IMAP IDLE loop + worker threads)
  • verdictmail-web.service — this Flask admin interface (port 80)
